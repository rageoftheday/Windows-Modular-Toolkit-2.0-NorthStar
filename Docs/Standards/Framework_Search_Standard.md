# Framework Search Standard

Search should use metadata first and fall back to object files when metadata is incomplete.

Searchable fields:

- Name
- Description
- Type
- Category
- Keywords
- Source

Search sources for Framework 2.0 Phase 36:

- Metadata index
- Framework tools
- User modules
- Workspace items
- Installer records
- Documentation

Search results should be paginated using the Framework Pagination Standard.
