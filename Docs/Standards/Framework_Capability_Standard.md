# Framework Capability Standard

A visible tool should exist only when it provides a unique user-facing capability.

If a feature extends an existing capability, add it as a mode, view, filter, or submenu inside the existing Center.

Actions:

- KEEP: unique capability or primary center.
- MERGE: capability should move into an existing center.
- RETIRE: obsolete and no capability loss.
- REVIEW: human review needed before action.
