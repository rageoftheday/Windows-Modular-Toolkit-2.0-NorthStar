# Framework Display Auto-Fit Standard - RETIRED

Dynamic window auto-fit was tested and removed before release.

Reason:
- Caused cursor/input placement issues in Windows Terminal.
- Caused inconsistent window behavior across hosts.
- Added more risk than value.

Current standard:
- Do not automatically resize the console window.
- Let users resize Windows Terminal/PowerShell manually.
- Future display options may use manual Compact/Standard/Large profiles only.
