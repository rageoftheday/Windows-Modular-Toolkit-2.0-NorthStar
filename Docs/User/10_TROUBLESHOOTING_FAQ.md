# Troubleshooting & FAQ

## What is Incoming?
Incoming is the universal drop folder. Put files there before scanning/importing.

## What is Repository?
Repository is managed storage for imported files.

## What is a module?
A module is a toolkit-launchable user tool with metadata.

## What is Detection Library?
Detection Library teaches the framework how to identify content by extension, indicators, or folder structure.

## Do I need Git?
Only for clone workflows. Source ZIP and release downloads can work without Git.

## Can I run from USB?
Yes. The toolkit is designed to be portable and USB-friendly.

## Why did my file go to Custom?
It matched a custom detection or learned format instead of a protected repository type.

## How do I reset formats?
Use Metadata Library Manager -> Repository Formats or Library Recovery.

## How do I reset detections?
Use Metadata Library Manager -> Detection Library or Library Recovery.

## How do I fix broken repository folders?
Use Repository Builder / Repair Repository.

## Why is Q not shown everywhere?
Q is only for the Main Menu. B goes back one level from submenus.

## What if I see red text?
Check Logs\errors.log and Logs\transcript.log if available. Reproduce the issue and report the exact screen.
