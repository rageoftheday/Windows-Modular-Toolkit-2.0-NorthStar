# Search & Discovery Guide

## What Is It?
Search Center finds modules, repository items, documentation, workspace links, framework resources, keywords, and detected content.

## Why It Exists
Users should not need to remember where every tool lives.

## How Search Gets Smarter
Search uses:
- module names
- descriptions
- categories
- subcategories
- keywords
- repository records
- learned formats
- Detection Library results
- documentation

## How To Improve Search Results
1. Add better keywords in Metadata Library Manager.
2. Add or fix categories/subcategories.
3. Add learned repository formats.
4. Add Detection Library rules.
5. Re-scan or rebuild search/cache if needed.

## Examples
Search python to find Python projects or modules.
Search zip to find archives.
Search photoshop to find Photoshop detections.
Search ps1 to find scripts.

## Recovery
If search stops finding expected items:
1. Rebuild toolkit cache/search if available.
2. Re-scan repository.
3. Check metadata libraries.
4. Run validation/audit tools.
