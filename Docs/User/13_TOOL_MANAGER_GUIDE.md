# Tool Manager Guide

## What Is It?
Tool Manager manages existing user tools and modules.

## Use It To
- organize tools
- rename user tools
- move user tools
- hide user tools
- remove user tools
- scan module content

## Safety Rule
Framework components are protected and should not be deleted through Tool Manager.

## Recovery
If a user tool is missing, check trash/history if enabled, backups, module folders, and validation tools.
