# Getting Started

## What Is The Toolkit?
Windows Modular Toolkit is a portable framework for organizing, managing, validating, and running modular tools.

It is USB-friendly and does not require an installer.

## Core Idea
- The framework manages itself.
- Users manage the content.

Framework components are protected. User modules, repository content, categories, installers, websites, folders, and workspace links are user-managed content.

## Main Menu Areas
[S] Search Center
Find tools, modules, documentation, repository items, workspace links, and framework resources.

[L] Launch Center
Browse and launch toolkit modules, software deployments, installers, and utilities.

[W] Workspace Center
Open personal toolkit shortcuts and linked items.

[D] Dashboard Center
View toolkit status, health summaries, statistics, recommendations, and recent activity.

[F] Framework Center
Maintain, repair, audit, and configure the framework itself.

[M] Module Tool Manager
Create, validate, package, repair, and maintain modules.

[R] Repository Manager
Scan Incoming, organize Repository items, and manage software deployments.

[T] Tool Manager
Organize, rename, move, clone, hide, remove, scan, and manage existing tools.

[H] Help Center
Documentation, tutorials, troubleshooting, standards, architecture, and guidance.

## Basic Navigation
B = Back one level from any submenu.
Q = Quit from the Main Menu only.

## First 10 Minutes
1. Open Help Center.
2. Read this guide.
3. Open Repository Manager.
4. Put a test file into Incoming.
5. Run Scan Incoming.
6. Search for the item in Search Center.
7. Read How To Make A Module before creating real tools.
