# Workspace Center Guide

## What Is It?
Workspace Center stores personal shortcuts and linked items.

## Important Rule
Workspace links by default. It does not move or copy modules unless a specific workflow says so.

## Use It For
- quick links
- work profiles
- frequently used resources
- personal toolkit shortcuts

## Recovery
If a workspace item is missing, check Config\workspace_items.json and workspace templates. Restore from backup if needed.
