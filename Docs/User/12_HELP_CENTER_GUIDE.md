# Help Center Guide

## What Is It?
Help Center is the built-in documentation browser and support area.

## Path
Main Menu -> [H] Help Center

## What It Contains
- Help & Support
- How To Guide
- Docs folder access
- Logs folder access
- Test log viewer

## How To Find Any Document
Main Menu -> [H] Help Center -> [1] Help & Support -> [13] All Documents

## If A Document Is Missing
Open Docs folder and verify it exists.
If it ships with the toolkit, it should also be reachable from All Documents.
