# Workspace Guide

Workspace is your personal area inside the Windows Modular Toolkit.

Use Workspace for quick access to common modules, work files, folders, scripts, websites, notes, and shortcuts.

## Workspace Center

Open and launch workspace items.

## Workspace Manager

Create templates, add items, sort incoming files, and create sections.

## Section Order

The first three sections always appear first:

1. Quick Access
2. Work
3. Personal

All other sections are shown alphabetically.

## Module Links

Module links do not copy modules. They point to the original toolkit module, so updates to the module are automatically used by Workspace.

## Selection Over Entry

If the framework knows the available choices, it will show a numbered selection instead of asking you to type internal paths or object references.
